# 自用 ImmortalWrt X86_64  
* 基于 ImmortalWrt 最新源码每日自动编译
* 支持在线升级，在luci界面或者SSH界面可以自动拉取最新版本进行升级  
  
## 修改设置  
* 分区大小默认1600M
* 开启 IPv6 功能

## 包含插件  
* DDNSTO 远程控制
* Wechatpush 微信推送
* OpenClash（dev版本主程序和内核）
* KMS 服务器
* Alist
* DDNS-GO
* frp 服务器
* UPnP
